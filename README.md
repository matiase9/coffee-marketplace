

Requirements
------------

* [Composer][2].
* [Docker][3].

Installation
-------------

* Run `docker-compose up -d`.
* Next, `composer install`
* Add the next line in the hosts file (/etc/hosts)
```
    127.0.0.1       local.coffee-marketplace.com
```  

* Check the domain [local.coffee-marketplace.com][4].
